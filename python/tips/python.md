## tips

+ `a // b` 整除
    + `a / b` 产生小数

+ `nonlocal`
    + 嵌套函数中修改外部函数的变量

+ `[1]*10` 扩展数据

+ range vs xrange（Python2）
    + range
        + 直接生成列表占用较大内存
    + xrange
        + 返回生成器
